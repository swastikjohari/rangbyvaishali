# Rang by Vaishali 🎨

An aesthetic, minimal e-commerce website for original handmade paintings by Vaishali.

**Live Site:** [Coming Soon]

---

## About

"Rang" means colour in Hindi. This gallery showcases Vaishali's original acrylic paintings — from serene seascapes and vibrant landscapes to expressive portraits and spiritual art inspired by Indian mythology.

Each painting is one of a kind, handmade with love.

---

## Features

- Minimal, gallery-style design with warm earthy tones
- Responsive layout (works on desktop, tablet, and mobile)
- Interactive gallery with category filters (Portrait, Landscape, Spiritual)
- Add-to-cart with animated slide-up cart drawer
- Scroll-triggered reveal animations
- 3D tilt effect on painting cards
- Floating particles and animated colour blobs in the hero
- Paint splatter effect on click
- Typing animation for hero subtitle
- Price counter animation on scroll
- Confetti burst on form submission
- Contact form for custom commissions

---

## Tech Stack

- HTML5
- CSS3 (custom properties, grid, animations)
- Vanilla JavaScript (no frameworks or dependencies)
- Google Fonts (Cormorant Garamond + Inter)

## Getting Started

1. Clone the repo
2. Open `index.html` in your browser
3. No build step or dependencies required

---

## Deployment

This is a static site — deploy anywhere:

- **Netlify**: Drag & drop the folder at [app.netlify.com](https://app.netlify.com)
- **GitHub Pages**: Push to repo → Settings → Pages → Deploy from main
- **Vercel**: Connect your GitHub repo at [vercel.com](https://vercel.com)

---

## Customisation

- Replace images in `images/` with new paintings
- Edit painting titles, prices, and categories in `index.html`
- Adjust colours in the CSS `:root` variables
- Connect the contact form to [Formspree](https://formspree.io) or [EmailJS](https://emailjs.com)
- Add Razorpay/Stripe for payment processing
